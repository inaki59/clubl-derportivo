# proyect django + react
Welcome to my web project made with react on the frontend and django on the backend


# download project 
 * create folder 
 * cd folder 
 *   `git clone https://github.com/inaki59/clubl-derportivo.git master`

 ## dependencies in front
  `cd client2`
